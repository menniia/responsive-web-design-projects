# freeCodeCamp Survey Form

This is a survey form project created as part of the freeCodeCamp Responsive Web Design Course. The survey form collects feedback from users regarding their experience with freeCodeCamp and their preferences.

## Features

- Collects user's name, email, age (optional), and current role.
- Asks whether the user would recommend freeCodeCamp to a friend.
- Allows users to select their favorite feature of freeCodeCamp.
- Provides checkboxes for users to indicate what improvements they would like to see.
- Offers a textarea for users to provide additional comments or suggestions.
- Submission button to submit the survey form.

## Usage

1. Fill out the required fields in the survey form.
2. Choose options from the radio buttons, dropdown, and checkboxes as applicable.
3. Add any additional comments or suggestions in the textarea.
4. Click the "Submit" button to submit the form.

## Technologies Used

- HTML5
- CSS3
